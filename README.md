# 🚗 Aventador Landing Page

A modern and responsive landing page inspired by the **Lamborghini Aventador**. This project showcases a clean UI, smooth animations, and a premium design focused on one of the world's most iconic supercars.

## 🌐 Live Demo

https://mahdimix1386-design.github.io/Aventador/

## ✨ Features

* Responsive design for desktop and mobile
* Modern and clean user interface
* Smooth animations and transitions
* High-quality images
* Fast loading static website
* Easy to customize

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/mahdimix1386-design/Aventador.git
```

Open the project folder and launch `index.html` in your browser.
