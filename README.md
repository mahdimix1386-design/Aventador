# 🚗 Lamborghini Aventador Landing Page

A modern and responsive landing page for the **Lamborghini Aventador**, built using HTML, CSS, and JavaScript.

## ✨ Features

- Modern UI Design
- Fully Responsive
- Smooth Scrolling
- Beautiful Animations
- Clean and Organized Code
- Fast Loading

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📸 Preview

Live Demo:
https://mahdimix1386-design.github.io/Aventador/

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/mahdimix1386-design/Aventador.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
